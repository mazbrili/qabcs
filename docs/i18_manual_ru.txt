+++ Инструкция по локализации алфавита +++

Чтобы добавить алфавит в программу qABCs, необходимо написать abc.json или abc.properties файл.
Образцы этих файлов вы можете найти в директории abcs.
abc.json позволяет детально описать все нюансы, а abc.properties больше нацелен на автоматизацию и лёгкость его написания.
Какой из этих форматов выбрать - переводчик решает сам. Программа поддерживает оба формата одинаково хорошо.

+++ Описание формата abc.properties +++
Глобальная переменная inheritsFrom описывает наследование языков, применяется для диалектов (см. ниже подробнее), она является опциональной. Формат: inheritsFrom: язык_родителя/abc.properties.
Глобальная переменная language описывает язык, обязательная переменная.
Глобальная переменная author описывает автора и адрес электронной почты, необязательная, но крайне желательная переменная.
Глобальная переменная espeak_params описывает параметры для espeak, необязательная переменная.
Глобальная переменная visible описывает видимость текущего файла в диалоге выбора языка, имеет значения true (видимый), false (невидимый); если она не указана, то файл является видимым.

misc:a=airplane==airplane=airplane
food:a=apple==apple
misc:а=АККОРДЕОН=АКОР-ДИ-ОН=accordion=accordion

первая переменная misc означает название игры, на данный момент в программе 6 игр: misc (в алфавитном порядке), food (продукты), animals (животные), instrument (музыка), toys (игрушки), а игра rand (в случайном порядке) описания не требует.
вторая переменная a - это буква алфавита (можете писать её в любом регистре); при наличии файла с таким именем в директории abcs/your_lang/sounds/alpha воспроизводится он, иначе, если он не существует, и нет наследования, то это произношение для espeak; для случая наследования в первую очередь ищется и играется файл из директории abcs/your_lang/sounds/alpha, если он отсутствует, но существует в директории родителя, то играется тот; если звукового файла нигде не найдено, то это произношение через espeak наследника.
третья переменная airplane - это отображаемое программой слово (можете писать его в любом регистре), при этом символы подчёркивания автоматически заменятся на пробелы.
четвёртая переменная АКОР-ДИ-ОН необязательная (можете писать её в любом регистре), она описывает произношение слова для espeak; в случае если espeak должен работать, но значение этой переменной не указано, то ничего не будет произнесено.
пятая переменная на анлийском языке описывает имя файла изображения из директории abcs/all/pics, а также имя файла звука из директории abcs/your_lang/sounds/words, или при наличии наследования имя звукового файла родителя из аналогичной директории, который используется, если у наследника звуковой файл отсутствует, но при этом он имеется у родителя; если звукового файла нет, и нет наследования, то если указано произношение в четвёртой переменной, то espeak произносит его, иначе ничего не произносится; если звукового файла нет, и есть наследование, то если существует звуковой файл в соответствующей директории родителя, то проигрывается он, иначе, если звуковой файл нигде не найден, то если указано произношение в четвёртой переменной, то espeak произносит его, иначе ничего не произносится.
шестая переменная опциональная, она описывает звуковой эффект (шум), является именем файла из директории abcs/all/noises.

+++ Описание формата abc.json +++
В разделе general указываются глобальные переменные.
Глобальная переменная inheritsFrom описывает наследование языков, применяется для диалектов (см. ниже подробнее), она является опциональной. Формат: "inheritsFrom" : "язык_родителя/abc.properties".
Глобальная переменная language описывает язык, обязательная переменная.
Глобальная переменная author описывает автора и адрес электронной почты, необязательная, но крайне желательная переменная.
Глобальная переменная espeak_params описывает параметры для espeak, необязательная переменная.
Глобальная переменная visible описывает видимость текущего файла в диалоге выбора языка, имеет значения true (видимый), false (невидимый); если она не указана, то файл является видимым.
Глобальная переменная speak_method указывает каким методом (file или espeak) генерировать речь: через файл (file) или через espeak (espeak).

Глобальные переменные espeak_params, speak_method могут быть переопределены локальными переменными.

В разделе letters описываются всё. Он состоит из структур следующего вида:

	{"А": {
        "espeak_words": "А",
		"misc": {
		      "pic": "music/accordion.png",
		      "name": "АККОРДЕОН",
		      "espeak_words": "АКОР-ДИ-ОН",
		      "noises": "accordion.ogg"
		},
		"food": {
		      "pic": "food/watermelon.png",
		      "name": "АРБУЗ",
		      "espeak_words": "АР-БУЗ"
		},
		"animals": {
		      "pic": "animals/stork.png",
		      "name": "АИСТ",
		      "espeak_words": "АИСТ",
		      "noises": "stork.ogg"
		},
		"instrument": {
		      "pic": "music/harp.png",
		      "name": "АРФА",
		      "espeak_words": "АРФА",
		      "noises": "harp.ogg"
		},
		"toys": {
		      "pic": "toys/watercolor.png",
		      "name": "АКВАРЕЛЬ",
		      "espeak_words": "АКВА-РЕЛЬ"
		}
	}},

или:

	{"A": {
		"sound_letter": "A.ogg",
		"misc": {
		      "pic": "misc/airplane.png",
		      "name": "AIRPLANE",
		      "sound_pic": "airplane.ogg",
		      "noises": "airplane.ogg"
		},
		"food": {
		      "pic": "food/apple.png",
		      "name": "APPLE",
		      "sound_pic": "apple.ogg"
		},
		"animals": {
		      "pic": "animals/angelfish.png",
		      "name": "ANGELFISH",
		      "sound_pic": "angelfish.ogg"
		},
		"instrument": {
		      "pic": "music/accordion.png",
		      "name": "ACCORDION",
		      "sound_pic": "accordion.ogg",
		      "noises": "accordion.ogg"
		},
		"toys": {
		      "pic": "toys/anklet.png",
		      "name": "ANKLET",
		      "sound_pic": "anklet.ogg"
		}
	}},

на данный момент в программе 6 игр: misc (в алфавитном порядке), food (продукты), animals (животные), instrument (музыка), toys (игрушки), а игра rand (в случайном порядке) описания не требует.
переменная А - это буква алфавита (можете писать её в любом регистре).
переменная sound_letter описывает звуковой файл в директории abcs/your_lang/sounds/alpha, он воспроизводится при наличии; для случая наследования в первую очередь ищется и играется файл из директории наследника abcs/your_lang/sounds/alpha, если он отсутствует, но существует в директории родителя, то играется тот.
переменная espeak_words описывает произношение через espeak; при наследовании используется произношение, указанное у наследника в приоритете над родителем.
если нет наследования, и если не указан звуковой файл (или он не существует), не указано произношение через espeak, то ничего не произносится.
переменная name - это отображаемое программой слово (можете писать его в любом регистре), при этом символы подчёркивания автоматически заменятся на пробелы.
переменная pic описывает имя файла изображения из директории abcs/all/pics.
переменная sound_pic описывает имя файла звука из директории abcs/your_lang/sounds/words, или при наличии наследования имя звукового файла родителя из аналогичной директории, который используется, если у наследника звуковой файл отсутствует, но при этом он имеется у родителя.
переменная noises опциональная, она описывает звуковой эффект (шум), является именем файла из директории abcs/all/noises.

+++ Общие замечания +++
При составлении файла важно расположить буквы в алфавитном порядке.
Все имена файлов программы в нижнем регистре, вы можете писать в любом регистре, программа автоматически будет использовать нижний регистр; указание расширения не требуется, при этом в имени файла используется знак подчёркивания вместо пробела.
